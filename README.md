# reth_miner
reth miner

###  download code
``` git clone https://github.com/huaigu/reth_miner```

### install dependency
```npm i ```

### edit .env
rename .env.example to .env then
Fill `RPC_URL` and `PrivateKey`

### run

For Ethereum: 
`node main.js`

For Arb: 
Mint amount was set to 100 by default(you can change it in .env file)
`node main_arb.js`
